# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for the Admin Chat & Audit system.

## Template

Use the following template for new ADRs:

```markdown
# ADR-XXX: [Title]

**Status:** [Proposed/Accepted/Rejected/Superseded]
**Date:** YYYY-MM-DD
**Deciders:** [Names]

## Context
[What is the issue we're motivating to solve?]

## Decision
[What is the change we're proposing/have agreed to?]

## Consequences
[What becomes easier or more difficult to do because of this change?]
```
